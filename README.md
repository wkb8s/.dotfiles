# dotfiles
available in macOS and Ubuntu

## Installation
```
git clone https://github.com/wkb8s/dotfiles.git
cd dotfiles && ./main
```
## Manual settings
### 1Password
```
(Developer) -> (SSH Agent)
```

### neovim
```
:PlugInstall
```

### tpm
```
(prefix) + I
```

### System Settings
- reduce motion

- Bluetooth
    - automatically connected after setting up Apple ID

### Vivaldi
- hide tool bar
```
(option) + (command) + P
```

- enable CSS modifications
```
vivaldi://experiments/
```

### AtCoder
```
acc login
oj login https://atcoder.jp/home
```
